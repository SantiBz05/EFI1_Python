## Clonar el proyecto

- ```git clone https://github.com/SantiBz05/EFI1_Python.git```

- ```cd EFI1_Python```
##### Crear el entorno virtual
- ```python3 -m venv env```
##### Activar el entorno virtual
- ```source env/bin/activate```
##### Instalar requerimientos
- ``` pip install -r requirements.txt```
##### Correr el proyecto
- ```flask run --reload```
